# Data-Analysis-with-R-A-project
This project reflects my hard learning journey in data analysis with R. This project focuses on the analyzing and visualizing the dataset to get insights from the sales data in an electronic company named Gorgor Electronic.
This project inlcudes key steps necessary for data exploration.
A. Which product generated the most revenues across all the other products.
B. Whiich Month the sales revenue was peak and which the sales went down.
C. And if there is relationship between sales reveneue generated each month.

The project attached with R codes and some visualizations to aid the general view of the datasets.
